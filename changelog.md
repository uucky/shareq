# ShareQ KTV 点歌系统 - 更新日志 (Changelog)

ShareQ 是一个专为多人聚会设计的 KTV 在线点歌小程序/网站。它支持创建/加入房间、歌手昵称/头像设置、歌手自定义本地图片头像上传与网格折叠滚动、歌曲点播与链接添加、歌曲置顶、历史已唱记录回看，以及主持人/房管的管理特权（踢人、切歌、撤销/重做）。此外，系统提供了精美的亮色模式切换、消息通知历史、会话数据统计、切歌提醒音效、点歌者自主切/删歌，及角落若隐若现的交互式导出“彩蛋”存档功能。

## [v1.3.0] - 2026-06-06 18:55:00
### 新增 (Added)
- **自定义头像上传与优化**：
  - 允许歌手在登录大厅和个人中心上传本地图片，使用 Canvas 自动进行 128×128 圆形居中裁剪与压缩（Base64 编码传输）。
  - 对头像选择区域设置最大高度与自定义细滚动条，防止头像选项过多拉长页面。
  - 将默认头像网格中的 Emoji 数量扩充为 70 余个更具个性的头像选项。
- **轮到演唱弹窗提醒**：
  - 当播放列表切换且轮到当前成员的歌曲播放时，系统会使用 Web Audio 合成双音 chime 提示音，并在屏幕中央弹出精美的"轮到你上台啦"提醒卡片（6 秒自动消失或手动关闭）。
- **点歌者自主切/删歌权限**：
  - 当前正在演唱的歌曲若为某成员自己所点，该成员无需房管权限也可直接点击"切歌"按钮切走自己的歌。
  - 普通成员可删除自己在排队中点播的歌曲；不属于自己的歌曲只有房主/房管才能删除。
- **隐藏彩蛋入口调整**：
  - 将导出存档功能的彩蛋入口由双击人数标签，改为左下角一个若隐若现的常驻圆形微图标（支持亮暗色主题自适应）。
  - 点击 lobby 或 room 顶部的 ShareQ 品牌 LOGO，将展示悬浮小提示："🎵 ShareQ 共享卡拉OK歌单系统 by uucky"。
- **更多菜单新增"关于 ShareQ"**：
  - 在右上角三点更多菜单中加入"关于 ShareQ"选项，点击后弹出精美的 About 卡片，展示作者、版本号（v1.3.0）、技术栈及简介。
- **Render 一键部署支持**：
  - 新增了 `render.yaml` 部署配置文件。
  - 在 `README.md` 中添加了 Deploy to Render 徽章，实现极简的免配置一键免费部署。

### 修复 (Fixed)
- 修复了右下角消息历史铃铛按钮在 Hover 悬浮时变成透明的视觉缺陷（亮暗色模式均已修正）。
- 修复了所有使用 Base64 图片自定义头像在以下位置的渲染逻辑（全部由文本回退为图片正常显示）：
  - 顶部头像重叠堆栈、侧边成员列表、Now Playing 点歌人显示、待播歌单、已唱历史列表、个人资料编辑弹窗。

## [v1.0.0] - 2026-06-06 01:30:00
### 新增 (Added)
- **基础 KTV 业务功能**：支持通过 5 位随机房间号创建或输入房间号加入房间，支持自定义歌手昵称（提供随机昵称）和头像选择。
- **点歌与队列控制**：支持点播歌曲、可选歌手名和外部伴奏/MV链接，具备播放列表的“优先置顶”和“打乱歌单”功能。
- **权限角色机制**：引入主持人与房管（可由主持人转让或任命）特权机制，房管/主持人支持控制上一首、下一首、撤销和重做列表，并具备踢人特权。
- **互动特效与声音合成**：支持发送礼物（玫瑰、掌声、鸡蛋、拖鞋）的声效合成与浮动动画。
- **彩蛋导出存档**：双击人数标签或品牌 LOGO 可开启彩蛋，下载包含当天日期时间、多时区选择及列表排序的 HTML 交互式会话存档。
